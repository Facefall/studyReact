使用淘宝镜像[href="http://npm.taobao.org/"],设置cnpm
npm install -g cnpm --registry=https://registry.npm.taobao.org

使用脚手架来开发app
cnpm i create-react-app -g

之后创建项目
create-react-app 项目名
(项目名不要大写)

安装完毕后
去到项目文件夹下
yarn start

官网推荐用yarn包管理器
```
npm i yarn -g
yarn add jquery     -> cnpm i jquery
yarn start          -> cnpm start
yarn (默认安装依赖)   -> cnpm i
